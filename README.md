# training

Projet de formation de deux mois.

Création d'une application de gestion de parc informatique. 
Gestion CRUD par web services, application web et interface en ligne de commande

Formation sur java et sur les frameworks les plus courants. La formation consiste en un développement itératif qui reprend étape par étape les anciens frameworks et les remplace petit à petit dans le but de bien comprendre l'utilité et l'avantage de chacun d'entre eux. Une code review est mise en place à chaque étape permettant ainsi de corriger directement son code et de connaître les bonnes méthodes sans accumuler les erreurs. Nous avons effectué la mise en oeuvre du mode projet et de la méthodologie scrum et participé à des conférences tout au long de la formation sur les technologies comme Git, Maven, ou encore les Web services. Une présentation finale devant un product owner a clôturé la formation.

Environnement : Git, Maven, Spring, Hibernate, Java/JEE, Jenkins, Docker, WebService.
